The bar rhythmic position object

Type: Dictionary

Describes the rhythmic position of a notational object within a piece of music. This is comprised of a bar index plus a rhythmic position within that bar.

Keys:

Name Type Required? Description
"bar" measure number object Yes The bar number.
"position" rhythmic position object Yes The position within the bar.

Examples

This object is used in the following examples:

Jumps (D.S. al Fine), Multiple layouts, Octave shifts (8va), Organ layout
